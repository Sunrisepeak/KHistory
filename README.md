# KHistory
KHistory 是一个**优雅**的按键 检测/历史记录 工具

[**English**](README.en.md)



### 无需安装单文件下载即用

> **点击右边release获取对应平台最新版本**



## 一、功能特性

- 键盘按键检测
- 游戏手柄按键检测
- 控制
  - 按键历史记录数
  - 透明度
  - 检测帧率
- 游戏按键可视化(待优化)
